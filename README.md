# escritorio_academico_front
Escritorio Académico es una plataforma web diseñada para la gestión y administración de instituciones educativas. Este proyecto proporciona herramientas intuitivas y eficientes para gestionar estudiantes, docentes, ciclos escolares, cuotas, actividades extracurriculares y mucho más, facilitando la operación diaria de los colegios y escuelas.
